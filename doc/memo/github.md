# git init ~ git push
1. `git init`
2. `git git remote add origin <URL>`
3. `git add --all`
4. `git commit -m "first commit"`
5. `git push -u origin master`

# git add ~ git push
1. `git add --all`
2. `git commit -m "COMMENT"`
3. `git push -u origin master`

# 現在いるレポジトリのトップディレクトリの絶対パスを返す
1. `git rev-parse --show-toplevel`
